# Reference Copies

This folder contains cold email examples for AI-powered email generation.

## Files

| File | Purpose |
|------|---------|
| `TEMPLATE.md` | Blank template - copy-paste to create new emails |
| `examples.md` | Example emails showing different styles (human-readable) |
| `emails.json` | Same examples in JSON format (for scripts/AI) |

## Quick Start

1. Copy this entire `reference_copies/` folder to your client's campaign folder
2. Edit `examples.md` with your own copy (use `TEMPLATE.md` as guide)
3. Run the email copywriter to generate personalized sequences

## Creating Your Own Copies

1. Open `TEMPLATE.md`
2. Copy the entire template
3. Paste into your client's reference copies file
4. Fill in your subject line and body
5. Repeat for each email style you want

## Email Positions

| Position | When Sent | Purpose |
|----------|-----------|---------|
| `initial` | Day 0 | First touch - grab attention |
| `followup_1` | Day 2-3 | Add value, case study |
| `followup_2` | Day 5-7 | Social proof, direct ask |
| `followup_3` | Day 8-10 | Give value, soft close |
| `breakup` | Day 10-14 | Final email, close the loop |

## Placeholders

| Placeholder | Description |
|-------------|-------------|
| `{{first_name}}` | Lead's first name |
| `{{last_name}}` | Lead's last name |
| `{{company}}` | Company name |
| `{{title}}` | Job title |
| `{{industry}}` | Industry |
| `{{icebreaker}}` | Personalized intro (from enrichment) |
| `{{sender_name}}` | Your name |
| `{{sender_company}}` | Client company name |

## Tips

1. **Keep it short** - 100-150 words for initial, shorter for follow-ups
2. **One CTA per email** - Don't confuse the reader
3. **Add real metrics** - "35% reduction" beats "significant improvement"
4. **Test different styles** - Some audiences respond to casual, others to professional
